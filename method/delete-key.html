delete(key)

Deletes the value for a given key. Returns whether the key was found and successfully deleted.

On collections

  • Map
  • Dict
  • FastMap
  • LfuMap
  • LruMap
  • MultiMap
  • SortedMap
  • SortedArrayMap
  • GenericMap

Examples

var map = new Map({a: 10});
map.toObject();
map.delete("a");
map.toObject();
map.delete("a");
