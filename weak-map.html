WeakMap

A map of object keys to values with good garbage collection behavior. See WeakMap.

Examples

❯ var Dict = require("collections/dict");
❯ var dict = new Dict();
❯ var dict = new Dict(null, function (key) {
    return this.get(key.slice(0, key.length - 1));
});
❯ dict.set("a", 10);
true
❯ dict.get("abcdefg")
10
